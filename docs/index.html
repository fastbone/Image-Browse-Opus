⚡ Features

Everything you need.
Nothing you don't.

Built from the ground up for speed and simplicity. No cloud accounts, no subscriptions, no telemetry. Just your files, instantly.

Virtualized Gallery

Scroll through thousands of thumbnails without a hiccup. A virtualized layout renders only what is on screen, keeping memory low and performance high.

Fullscreen Viewer

Immersive distraction-free viewing with smooth zoom, pan, EXIF info overlay, filmstrip navigation, and crossfade transitions between images.

Video Playback (Windows)

On Windows, an integrated LibVLC player: seek, speed from 0.25x to 4x, zoom with mini-map, and 16 video formats alongside your images. macOS and Linux builds are image-first; video features are evolving.

Cross-platform

Native WPF on Windows; Avalonia on macOS and Linux. One shared core powers the same gallery, viewer, ratings, tags, and SQLite-backed cache everywhere you work.

Organization

Rate images 1–5 stars, tag favourites, and sort by name, date, size, dimensions, type, or rating. Per-folder sort preferences persist automatically.

Massive Format Support

From everyday JPEGs to camera RAW, HEIC, AVIF, JPEG XL, PSD, SVG, HDR, EXR, and more. Magick.NET everywhere; Windows adds a WPF fast-path for common types, Avalonia uses native decode for common rasters plus Magick.NET.

Instant Performance

SQLite-backed thumbnail cache with content-hash deduplication. Prescan entire folder trees in the background so every revisit is instant.

📸 Fullscreen Viewer

Immersive viewing experience

Zoom, pan, browse EXIF data, and navigate your collection without leaving fullscreen. The filmstrip keeps context always within reach.
